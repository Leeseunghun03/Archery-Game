# Archery Game (2 Players)

## Using Tool
Qt(C++)

## How to Play
1. Push 'Start' Button (Start Moving Lasers)
2. Push 'Stop' Button (Shoot Arrow)
3. Player1 repeat this Step 10 times
4. Change Player2 and repeat
